## PM 已定决策（用户已授权全权决策，依据"用户体验好 / 符合软件工程标准 / 长期易维护"原则）

1. **UI 默认端口 8080 → 7800**。8080 是最常见的冲突端口之一；7800 与 frp 自身端口族（7000/7400/7500）同段，便于记忆、不与上述四者重叠、且属于不常用端口，冲突概率低。
2. **UI 默认绑定地址 127.0.0.1 → 0.0.0.0**。frp_easy 本质是远程内网穿透管理工具，运维场景天然需要从其他设备访问 Web UI；项目已自带完整认证（argon2id + session + CSRF + 限流），0.0.0.0 不等于"无认证暴露"。配套要求：把 main.go 原有的"偏离 127.0.0.1 的 WARN"重构为**面向新默认值的简明安全提示**（提醒尽快完成 setup、说明可改回 127.0.0.1 仅本机访问），并在文档中清楚说明该取舍与加固方式。
